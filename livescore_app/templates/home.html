Livescore App


This app was developed by Felix Salcher and Marc Schl√∂mmer as part of the Information Systems Development Seminar.

The idea is that this app can be used to keep track of the scores for table tennis games as they are happening live and broadcasting this score to everyone who is interested in it.

For this purpose we have set up a tournament / game / player structure. The main focus point of this app is the websocket functionality implemented with django-channels that allows for live data connections to push the updated scores to the client.
